## ¿Qué es Backstage?

[Backstage](https://backstage.io/) es un marco de código abierto para construir portales para desarrolladores. Impulsado por un catálogo de software centralizado, Backstage restaura el orden en sus microservicios e infraestructura y permite a sus equipos de producto enviar código de alta calidad rápidamente sin comprometer la autonomía.

Backstage unifica todas sus herramientas de infraestructura, servicios y documentación para crear un entorno de desarrollo optimizado de extremo a extremo.

![catálogo de software](docs/assets/header.png)

De fábrica, Backstage incluye:

- [Catálogo de software de Backstage](https://backstage.io/docs/features/software-catalog/) para administrar todo su software, como microservicios, bibliotecas, canalizaciones de datos, sitios web y modelos de ML.
- [Plantillas de software de Backstage](https://backstage.io/docs/features/software-templates/) para poner en marcha rápidamente nuevos proyectos y estandarizar sus herramientas con las mejores prácticas de su organización.
- [TechDocs de Backstage](https://backstage.io/docs/features/techdocs/) para facilitar la creación, el mantenimiento, la búsqueda y el uso de documentación técnica, utilizando un enfoque de "documentación como código".
- Además, un ecosistema creciente de [complementos de código abierto](https://github.com/backstage/backstage/tree/master/plugins) que amplían aún más la personalización y la funcionalidad de Backstage.

Backstage fue creado por Spotify, pero ahora está alojado por la [Cloud Native Computing Foundation (CNCF)](https://www.cncf.io) como un proyecto de nivel de Incubación. Para obtener más información, consulte el [anuncio](https://backstage.io/blog/2022/03/16/backstage-turns-two#out-of-the-sandbox-and-into-incubation).
