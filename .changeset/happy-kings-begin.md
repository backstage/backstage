Add experimental support for ESLint flat-config format to `@backstage/eslint-plugin` and declare support for ESLint v9. Minimum ESLint version is now 8.57.1.

This is only pre-requisites to an ESLint version or config format migration.
This does **NOT** mean that the `@backstage/cli` is compatible with ESLint flat-config format.
This does **NOT** mean that the `@backstage/cli` is compatible with ESLint v9.
