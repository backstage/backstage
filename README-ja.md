## Backstage とは？

[Backstage](https://backstage.io/)は、開発者ポータルを構築するためのオープンソースフレームワークです。一元化されたソフトウェアカタログを搭載した Backstage は、マイクロサービスとインフラストラクチャの秩序を回復し、製品チームが自律性を損なうことなく高品質のコードを迅速に出荷できるようにします。

Backstage は、すべてのインフラストラクチャツール、サービス、およびドキュメントを統合して、エンドツーエンドで合理化された開発環境を作成します。

![ソフトウェアカタログ](docs/assets/header.png)

Backstage には、すぐに使える次の機能が含まれています。

- マイクロサービス、ライブラリ、データパイプライン、Web サイト、ML モデルなど、すべてのソフトウェアを管理するための[Backstage ソフトウェアカタログ](https://backstage.io/docs/features/software-catalog/)
- 新しいプロジェクトを迅速に立ち上げ、組織のベストプラクティスでツールを標準化するための[Backstage ソフトウェアテンプレート](https://backstage.io/docs/features/software-templates/)
- 「コードのようなドキュメント」アプローチを使用して、技術ドキュメントの作成、保守、検索、使用を容易にする[Backstage TechDocs](https://backstage.io/docs/features/techdocs/)
- さらに、Backstage のカスタマイズ性と機能をさらに拡張する[オープンソースプラグイン](https://github.com/backstage/backstage/tree/master/plugins)の成長するエコシステム

Backstage は Spotify によって作成されましたが、現在は[Cloud Native Computing Foundation（CNCF）](https://www.cncf.io)によってインキュベーションレベルのプロジェクトとしてホストされています。詳細については、[お知らせ](https://backstage.io/blog/2022/03/16/backstage-turns-two#out-of-the-sandbox-and-into-incubation)をご覧ください。
