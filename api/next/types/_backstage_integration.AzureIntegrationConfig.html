    Optionalcredentials

    credentials?: AzureDevOpsCredential[]

    The credentials to use for requests. If multiple credentials are specified the first one that matches the organization is used. If not organization matches the first credential without an organization is used.

    If no credentials are specified at all, either a default credential (for Azure DevOps) or anonymous access (for Azure DevOps Server) is used.
