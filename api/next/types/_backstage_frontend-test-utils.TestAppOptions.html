    OptionalmountedRoutes

    mountedRoutes?: { [path: string]: RouteRef<AnyRouteRefParams> }

    An object of paths to mount route ref on, with the key being the path and the value being the RouteRef that the path will be bound to. This allows the route refs to be used by useRouteRef in the rendered elements.

    Example

    renderInTestApp(<MyComponent />, {
      mountedRoutes: {
        '/my-path': myRouteRef,
      }
    })
    // ...
    const link = useRouteRef(myRouteRef)
