    createRoot

    • createRoot(element: Element): ComponentType<{ children?: ReactNode }>

      Creates the root component that renders the entire app.

      Parameters

      • element: Element

      Returns ComponentType<{ children?: ReactNode }>

      Remarks

      This method must only be called once, and you have to provide it the entire app element tree. The element tree will be analyzed to discover plugins, routes, and other app features. The returned component will render all of the app elements wrapped within the app context provider.

      Example

      export default app.createRoot(
        <>
          <AlertDisplay />
          <OAuthRequestDialog />
          <AppRouter>
            <Root>{routes}</Root>
          </AppRouter>
        </>,
      );
