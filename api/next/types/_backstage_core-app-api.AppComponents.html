    OptionalSignInPage

    SignInPage?: ComponentType<SignInPageProps>

    An optional sign-in page that will be rendered instead of the AppRouter at startup.

    If a sign-in page is set, it will always be shown before the app, and it is up to the sign-in page to handle e.g. saving of login methods for subsequent visits.

    The sign-in page will be displayed until it has passed up a result to the parent, and which point the AppRouter and all of its children will be rendered instead.
