    Type Alias PluginWrapperApiAlpha

    The Plugin Wrapper API is used to wrap plugin extensions with providers, plugins should generally use ExtensionBoundary instead.

    Remarks

    This API is primarily intended for internal use by the Backstage frontend system, but can be used for advanced use-cases. If you do override it, be sure to include the default implementation as well.
